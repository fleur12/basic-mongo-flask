# Basic MongoDB Flask Project

An interactive web application built with Flask and MongoDB, allowing users to add and manage data through a web interface.

---

## Features

- Connects to MongoDB Atlas for database management.
- Interactive HTML forms to add, view, and edit data.
- Simple, responsive UI with CSS styling.
- Basic CRUD (Create, Read, Update, Delete) operations.
